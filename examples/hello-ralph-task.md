# Example Task: Hello Ralph

## Task Overview

**Goal**: Create a simple "Hello World" script and verify it works.

**Why This Matters**: Tests that Ralph can create files, write code, and verify results.

**Success Indicator**: Script exists, runs successfully, and outputs "Hello, Ralph!"

---

## Success Criteria

- [ ] Create `hello-ralph.sh` script
- [ ] Script contains proper shebang and echo statement
- [ ] Script is executable (`chmod +x`)
- [ ] Running `./hello-ralph.sh` outputs "Hello, Ralph!"
- [ ] Script is committed to git

---

## Context

This is a simple test task to verify Ralph autonomous loop is working correctly.

**Tech stack**: Bash scripting

**Constraints**: Keep it simple - just a basic shell script.

---

## Notes

This task should complete in 1-2 iterations. If Ralph takes more than 5 iterations, something is wrong with the setup.
